# 贪吃蛇游戏 🐍

一个使用纯HTML、CSS和JavaScript开发的现代化贪吃蛇游戏。

## 游戏特色

- 🎨 **现代化界面设计** - 使用渐变背景和毛玻璃效果
- 🎮 **多种控制方式** - 支持键盘方向键和触屏滑动控制
- 📱 **响应式设计** - 适配移动设备和桌面设备
- 🏆 **分数系统** - 自动保存最高分记录
- ⏸️ **暂停功能** - 随时暂停和继续游戏
- 🚀 **动态难度** - 分数越高，游戏速度越快
- ✨ **视觉效果** - 精美的蛇身渐变和食物动画效果

## 如何游玩

### 桌面端控制
- **方向键** ↑↓←→ 控制蛇的移动方向
- **空格键** 暂停/继续游戏
- **暂停按钮** 点击界面上的暂停按钮

### 移动端控制
- **滑动屏幕** 在游戏区域滑动来控制蛇的移动方向
- **点击暂停按钮** 暂停/继续游戏

### 游戏规则
1. 🍎 控制蛇吃掉红色的食物来获得分数和增长身体
2. 🏃‍♂️ 每吃到一个食物得10分，游戏速度会逐渐加快
3. ⚠️ 避免撞到墙壁或蛇自己的身体，否则游戏结束
4. 🏆 尽可能获得更高的分数，挑战你的最高记录！

## 运行游戏

### 方法一：直接在浏览器中打开
1. 下载项目文件
2. 双击 `index.html` 文件
3. 游戏将在默认浏览器中打开

### 方法二：使用本地服务器
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx http-server

# 或使用PHP
php -S localhost:8000
```
然后在浏览器中访问 `http://localhost:8000`

## 技术特性

- **纯原生技术** - 不依赖任何第三方库
- **ES6+ JavaScript** - 使用现代JavaScript特性
- **Canvas绘图** - 使用HTML5 Canvas进行游戏渲染
- **本地存储** - 使用localStorage保存最高分
- **触摸支持** - 支持移动设备触摸控制
- **60FPS渲染** - 流畅的游戏动画效果

## 游戏截图预览

游戏采用深色主题配合霓虹色彩，营造现代游戏氛围：
- 蛇身采用青绿色渐变，头部有特殊高亮效果
- 食物设计为立体红苹果，带有高光效果
- 背景使用紫蓝渐变配合毛玻璃容器
- 游戏结束界面提供优雅的重启体验

## 开发说明

这个贪吃蛇游戏使用面向对象的设计模式，主要包含以下组件：

- `SnakeGame` 类 - 游戏主逻辑控制器
- 碰撞检测系统 - 处理边界和自碰撞
- 食物生成算法 - 随机生成不与蛇身重叠的食物
- 渲染引擎 - Canvas绘图和动画效果
- 输入处理 - 键盘和触摸事件监听
